**[1]** [시간제한]


[문제]

전자 점수 숫자판이 있다. 6개의 숫자를 나타낼 수 있고, 각 자리는 0 ~ 9 의 숫자를 표출할 수 있어 0 ~ 999999 사이의 수를 표현할 수 있다.

이 전자 숫자판은 쓰여진 숫자를 변경하기 위해서는 주어진 필터를 원하는 자리에 적용하여 변경하여야 한다. 필터의 길이는 1 ~ 6 이며, 필터에 쓰여지는 명령의 종류는 3가지이다.


0 명령은 해당 자리의 숫자를 변경하지 않는다.
+ 명령은 해당 자리의 숫자를 1 크게 만든다. 단, 쓰여져있는 숫자가 9인 경우는 이 명령을 수행할 수 없으므로, 필터를 적용하는 중 이런 경우가 발생한다면 필터를 사용할 수 없다.
- 명령은 해당 자리의 숫자를 1 작게 만든다. 단, 쓰여져있는 숫자가 0인 경우는 이 명령을 수행할 수 없으므로, 필터를 적용하는 중 이런 경우가 발생한다면 필터를 사용할 수 없다.

주어진 필터는 그대로 사용하거나, 뒤집어 사용할 수 있다.
필터를 사용할 때 필터의 일부가 전자 숫자판 밖에 위치하면 사용할 수 없다.

현재 전자 숫자판에 쓰여져 있는 수가 [159] 이고, 사용할 수 있는 필터가 [+-]이다. 그리고 필터를 사용하여 전자 숫자판의 수를 [555]로 바꾸고 싶다. 이런 경우 아래 그림과 같이 필터를 8번 적용하면 원하는 수로 변경이 가능하다. 그리고 이 방법으로 수를 변경하는 것은 필터를 사용하는 횟수가 최소인 경우이다.

다른 예시로, 현재 전자 숫자판에 적혀있는 수가 [1142]이고, 필터는 [+0-] 이다. 전자 숫자판의 수를 [350]으로 변경하려면 아래 그림처럼 필터를 3번 적용하면 되며, 이는 필터를 사용하는 횟수가 최소인 경우이다.


현재 전자 숫자판에 적혀진 수의 정보, 바꾸려는 수의 정보, 그리고 필터의 정보를 입력 받아, 필터를 최소로 적용하여 수를 변경하는 방법을 구하는 프로그램을 작성하시오.

키보드 입력은 sample_input.txt 파일로 대체합니다. 입력에 대한 테스트를 하려면 sample_input.txt 파일을 수정하세요.

[입력]
맨 처음 테스트 케이스의 개수 TT가 주어지며, 그 다음 줄부터 TT 개의 테스트 케이스가 주어진다. 각 테스트 케이스의 첫 줄에는 전자 숫자판에 적혀져 있는 수 AA (0 ≤ A ≤ 9999990≤A≤999999) 와 바꾸려는 수 BB (0 ≤ B ≤ 9999990≤B≤999999), 주어지는 필터의 길이 LL (1 ≤ L ≤ 61≤L≤6) 이 공백으로 구분되어 주어진다. 다음 줄에 필터의 정보가 입력된다. 필터의 정보는 +,-,0 세가지 명령으로 구성되어 있다.

```
5
159 555 2
+-
1142 350 3
+0-
116 676 2
+-
887 854 2
0-
370 994 2
+0
```

[출력]
각각의 테스트 케이스에 대하여 #x (x는 테스트 케이스 번호, 1부터 시작)을 출력하고 공백을 하나 둔 후 전자 숫자판의 수를 바꾸기 위해서 필요한 최소 필터 사용 횟수를 출력하시오. 만약 주어진 수로 변경이 불가한 경우에는 -1을 출력하시오.

```
#1 8
#2 3
#3 -1
#4 6
#5 12
```
